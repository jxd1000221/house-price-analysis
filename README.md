# 房价分析项目
# 上海市房地产数据分析平台

## 项目简介
基于上海链家房源数据，构建房地产市场多维度分析平台，实现房价、区域、户型等核心指标分析与可视化展示。

## 数据说明
- 数据量：约 10 万条房源数据
- 字段：价格、面积、户型、行政区、标签等

## 技术栈
- 数据处理：Python, Pandas
- 数据存储与分析：Hive, SQL
- 可视化：ECharts, Matplotlib
- 版本管理：Git + GitHub

## 项目目录
- `data/`：原始数据或虚拟数据
- `hive/`：Hive 建表及统计分析 SQL 脚本
- `python/`：数据清洗与分析脚本
- `visualization/`：可视化图表及截图

## 项目亮点
- 完整数据分析全流程：采集 → 清洗 → 分析 → 可视化
- 多维度分析房价、区域、户型、标签
- 图表支持交互筛选与数据钻取
